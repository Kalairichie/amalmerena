# Amal & Merena Wedding Invitation

Static wedding invitation website ready for GitHub and Render.

## Edit Details
Open `index.html` and edit the `window.INVITATION` block near the top:

- couple
- dateDisplay
- weddingDateISO
- time
- venue
- locationLink
- coverImage
- mainPhoto
- secondPhoto
- music
- message

## Render Deploy Settings
When creating a Render Static Site:

- Build command: leave blank
- Publish directory: `.`

If you upload this folder inside a larger repository, use this folder path as the publish directory instead:

`amal-merena-wedding-invitation`

## Files
- `index.html` - editable content and page structure
- `styles.css` - design and animation styles
- `script.js` - opening animation, countdown, scratch reveal, music, sharing
- `assets/` - photos and audio
